# RTSyn plugin

## Usage

### Update

Make sure you have last version of the dependencies:

```bash
xrepo update-repo
xmake require --upgrade
```

For development you may need to run:

```bash
xmake require --upgrade -fy <dependency_name>
```

### Compiling

For compiling:

```bash
xmake
```

### Tests

For running test:

```bash
xmake test
```
